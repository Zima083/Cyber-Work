# Cybersecurity Internship Task: OWASP Juice Shop

This repository contains the cybersecurity internship task submissions for Week 1, Week 2, and Week 3.

## Contents

- **Week_1**: Screenshots of vulnerabilities such as XSS and attempted SQL Injection.
- **Week_2**: Report describing security measures and conceptual fixes for the vulnerabilities.
- **Week_3**: Advanced security considerations, logging, and best practices.
- **Networking**: Section on networking and secure communication analysis.

## Tools Used

- OWASP Juice Shop
- Browser Developer Tools
- OWASP ZAP (for automated vulnerability scanning)
